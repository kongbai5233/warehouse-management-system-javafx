# 仓储管理系统（数据库应用实训）

本项目为本人大学期间《数据库应用实训》课程设计作业，主要实现一个基于数据库的简易仓储管理系统，用于完成课程要求及相关功能验证。  
项目内容以教学与课程设计为目的，适合作为学习与参考使用。

## 一、项目背景

随着信息技术的发展，传统的手工仓储管理方式已难以满足实际需求。  
本项目通过开发一套简单的仓储管理系统，实现对仓库配件信息及操作记录的管理，提高仓储管理的规范性和可操作性。

## 二、需求分析

系统主要实现以下功能：

- 用户账号管理（登录、注册）
- 仓库配件信息管理（增、删、改、查）
- 操作人员对配件的操作记录查询
- 基本的软件信息展示功能

## 三、功能模块

系统主要功能模块包括：

1. 主页面（登录 / 注册入口）
2. 登录页面
3. 注册页面
4. 登录成功后的管理主页面
5. 修改用户名功能
6. 仓库配件管理模块
7. 操作历史记录查询模块
8. 关于软件页面

## 四、数据库设计说明

系统数据库相关内容**已统一整理在 `data.sql` 文件中**，包括：

- 数据库表结构定义（建表语句）
- 系统初始化所需的基础数据
- 测试与演示所用的示例数据

使用本项目时，只需将 `data.sql` 导入数据库，即可完成系统所需的数据库环境初始化。

数据库主要包含以下表：

- 用户表  
- 仓库配件存储表  
- 历史操作记录表  

## 五、系统实现说明

- 登录与注册功能对账号和密码进行合法性校验，并与数据库进行交互
- 管理页面提供用户管理、仓储操作和历史查询等功能入口
- 所有仓储相关的增删改操作均会记录到历史操作表中
- 历史查询模块支持按操作人员查询历史记录
- 关于页面用于展示软件基本信息

- 本项目为课程设计作业，仅用于学习和存档  
- 数据库结构及初始化数据均存放于 `data.sql` 文件中
